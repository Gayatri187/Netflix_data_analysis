# Netflix Data Analysis

## Project Overview
This project performs exploratory data analysis on the Netflix Movies and TV Shows dataset to understand trends in content distribution, genres, ratings, and content growth over time.

## Dataset
The dataset contains information about Netflix titles, including movies and TV shows.

- Total records: 8,807
- Total features: 13

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights

### Movies vs TV Shows
Netflix contains significantly more movies than TV shows.

### Content Growth
The number of titles added to Netflix increased rapidly after 2015, reflecting the platform's expansion.

### Genre Distribution
Drama and International Movies are among the most common genres on Netflix.

### Ratings Distribution
TV-MA is the most common rating, indicating that Netflix primarily targets mature audiences.

## Visualizations
The project includes visualizations such as:

- Movies vs TV Shows distribution
- Top producing countries
- Content growth over time
- Genre distribution
- Rating distribution

## Conclusion
The analysis highlights Netflix's growth in content production and its focus on mature, globally diverse entertainment.
